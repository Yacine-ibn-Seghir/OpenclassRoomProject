A PROPOS DE MOI

Titualire d'un BTS électronique, mes expériences les plus marquantes ont été les phases de test dans une chambre à haute température (45° avec un taux d'humidité assez conséquant) sur un prototype de machine d'affranchissement. Les tests consistaient à passer de differentes type de plies et de vérifier si les impressions timbres étaient conformes aux cahiers des charges, ces tests s'étaient déroulé chez NEOPOST.

De même, j'ai efféctué des tests à haute et basse température sur des boitiers éléctronique permettant de faire fonctionner le système de climatisation automobile, ces tests se sont déroulés chez VALEO.
